# Who Should Use AgentFOS

AgentFOS is for builders who need autonomous agents to make financial decisions that are deterministic, explainable, auditable, and verifiable.

## Good Fits

- Agent developers building capital allocation workflows
- RWA applications that need policy-aware decisioning
- Protocol teams that want agent-readable risk and allocation outputs
- Financial automation teams that need deterministic risk scoring
- Pharos builders who need verifiable on-chain proof of agent actions

## Poor Fits

AgentFOS is not designed to be a general AI assistant, discretionary trading bot, portfolio tracker, or dashboard-only analytics app.

It is a financial decision layer.

## What It Gives Agents

| Need | AgentFOS Capability |
|---|---|
| Discover opportunities | Opportunity Skill |
| Enforce constraints | Policy Skill |
| Score risk | Risk Skill |
| Allocate capital | Allocation Skill |
| Prove decisions | Optional Pharos Proof |
