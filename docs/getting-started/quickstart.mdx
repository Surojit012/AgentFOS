# Quickstart

The fastest way to use AgentFOS is to call `/agent` with a natural-language capital request.

## Python

```python
import requests

response = requests.post(
    "https://agentfos.onrender.com/agent",
    json={
        "query": "Find low risk RWAs above treasury yields with $100k capital"
    }
)

print(response.json())
```

## cURL

```bash
curl -sS -X POST https://agentfos.onrender.com/agent \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Find low risk RWAs above treasury yields with $100k capital"
  }'
```

## Direct Allocation

Use `/allocate` when your agent already knows the capital amount and risk preference.

```bash
curl -sS -X POST https://agentfos.onrender.com/allocate \
  -H "Content-Type: application/json" \
  -d '{
    "capital": 100000,
    "risk": "medium",
    "min_apy_spread": 0
  }'
```

The response includes portfolio metrics, allocations, narrative, and on-chain status fields.
