# End-to-End Example

This example runs the natural-language agent path.

## Request

```python
import requests

response = requests.post(
    "https://agentfos.onrender.com/agent",
    json={
        "query": "Find low risk RWAs above treasury yields with $100k capital"
    }
)

print(response.json())
```

## Expected Response Shape

```json
{
  "query": "Find low risk RWAs above treasury yields with $100k capital",
  "steps": [
    {
      "step": "Parse Query",
      "status": "complete",
      "result": "Capital: $100,000, Risk tolerance: low"
    }
  ],
  "recommendation": {
    "capital": 100000,
    "risk_preference": "low",
    "allocations": []
  },
  "answer": "Analyzed RWA protocols and generated an allocation response."
}
```

The exact allocations depend on available data and risk filters.
