# Why AgentFOS

Autonomous agents can read, reason, and act. But financial decisions require more than reasoning.

They require policy, deterministic risk scoring, allocation logic, and verifiable records.

## The Gap

Most agent systems can explain a market. AgentFOS decides whether an agent should allocate capital under a known policy.

## The Difference

| Common Agent Tool | AgentFOS |
|---|---|
| Generates analysis | Produces structured decisions |
| Uses open-ended reasoning | Uses deterministic scoring |
| Gives a recommendation | Enforces policy and allocation constraints |
| Leaves weak audit trail | Can write proof on Pharos |

## The Principle

Every number should be traceable. Every decision should be explainable. Every proof should be verifiable.
