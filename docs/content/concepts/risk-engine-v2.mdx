# Risk Engine v2

AgentFOS uses deterministic risk scoring.

It does not use AI predictions, LLM reasoning, model inference, or hidden heuristics. Every score is computed from a replayable input snapshot.

## Dimensions

Risk Engine v2 scores five dimensions for a total of 100 points.

| Dimension | Max |
|---|---:|
| Liquidity | 25 |
| Protocol Maturity | 20 |
| Transparency | 20 |
| Concentration Risk | 20 |
| Yield Sustainability | 15 |

```text
risk_score = liquidity + maturity + transparency + concentration + yield_sustainability
```

## Rating Labels

| Score | Label |
|---|---|
| 90-100 | Excellent |
| 75-89 | Low Risk |
| 60-74 | Moderate Risk |
| 40-59 | High Risk |
| 0-39 | Very High Risk |

## Data Provenance

Every dimension reports data provenance.

| Provenance | Meaning |
|---|---|
| Live | Data fetched successfully |
| Degraded | Fallback, stale, partial, or static data used |
| Insufficient | Not enough usable data available |

AgentFOS never crashes when providers fail. It degrades gracefully and marks the score quality so consuming agents know what kind of artifact they are using.
