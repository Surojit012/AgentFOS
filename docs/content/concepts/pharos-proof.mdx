# Pharos Proof

AgentFOS can record the top allocation decision on Pharos Atlantic Testnet.

The on-chain proof is optional. It occurs when the backend has Pharos configuration, Foundry `cast`, a private key, and a deployed `AgentFOSSkill` contract address.

## What Gets Stored

The contract stores:

- caller address
- protocol key
- risk score
- APY basis points
- timestamp

It also stores latest risk score per protocol.

## Fee

`allocate(string,uint256,uint256)` charges exactly `0.001 PHRS`.

## Read Commands

```bash
cast call $AGENTFOS_CONTRACT_ADDRESS \
  "getRiskScore(string)(uint256)" \
  ondo \
  --rpc-url $RPC
```

```bash
cast call $AGENTFOS_CONTRACT_ADDRESS \
  "getLastAllocation()(address,string,uint256,uint256,uint256)" \
  --rpc-url $RPC
```
