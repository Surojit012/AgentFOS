# Policy Skill

The Policy Skill checks whether an agent action is allowed.

It enforces spend limits, risk tolerance, asset allowlists, and minimum risk score requirements.

## Endpoint

```text
POST /policy
```

## Request

```json
{
  "action": "allocate",
  "amount": 50000,
  "asset": "maple",
  "risk_tolerance": "low",
  "max_single_allocation": 0.6,
  "min_risk_score": 50
}
```

## Response

```json
{
  "allowed": true,
  "action": "allocate",
  "asset": "maple",
  "amount": 50000,
  "reason": "Policy PASS - Maple Finance approved for allocate.",
  "warnings": []
}
```

## What It Protects

- Spending above configured limits
- Unknown protocols
- Protocols below minimum risk score
- High-risk assets for low-risk agents
- Assets outside allowed risk tolerance
