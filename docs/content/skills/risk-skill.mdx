# Risk Skill

The Risk Skill scores a protocol deterministically.

It uses Risk Engine v2. There is no AI-generated risk score and no LLM reasoning in the scoring path.

## Endpoint

```text
POST /risk
```

## Request

```json
{
  "protocol": "ondo"
}
```

## Backward-Compatible Response

Existing consumers can continue to read top-level fields:

```json
{
  "protocol": "ondo",
  "protocol_name": "Ondo Finance",
  "risk_score": 81,
  "risk_level": "Low",
  "factors": {},
  "recommendation": "Ondo Finance scores 81/100 (Low Risk)."
}
```

## Risk Engine v2 Additions

Risk Engine v2 adds:

- `rating`
- `overall_confidence`
- `breakdown`
- `scored_at`
- `schema_version`

See [Risk Engine v2](/concepts/risk-engine-v2).
