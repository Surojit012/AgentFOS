# POST /opportunity

Fetches RWA yield opportunities with benchmark enrichment and risk scoring.

## Request

```json
{
  "risk_level": "medium",
  "min_apy": 0,
  "min_tvl": 0
}
```

## Request Fields

| Field | Type | Required | Description |
|---|---|---:|---|
| `risk_level` | string | No | `low`, `medium`, or `high` |
| `min_apy` | number | No | Minimum APY filter |
| `min_tvl` | number | No | Minimum TVL filter |
